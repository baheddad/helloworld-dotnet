# HelloWorldApp

A very simple "Hello World" ASP.NET Core (.NET 8) web app.

## Run locally
```bash
dotnet run
```
Then open http://localhost:5000

## Files
- `HelloWorldApp.csproj` — project file (.NET 8 web app)
- `Program.cs` — the whole app; serves a Hello World HTML page at `/`
